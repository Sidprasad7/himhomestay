## What It Is

A complete hospitality website for a homestay property — showcasing room categories, dining, gallery, and blog content with a clean, conversion-focused UI.

---

## Pages

| Page | Purpose |
|---|---|
| `index.html` | Homepage — hero, highlights, overview |
| `luxury.html` | Luxury room category |
| `standard.html` | Standard room category |
| `restaurant.html` | Dining and food offerings |
| `gallery.html` | Photo gallery |
| `blog.html` | Blog listing page |
| `contact.html` | Contact and enquiry form |
